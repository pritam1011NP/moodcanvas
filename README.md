## вњЁ Features

- **AI Emotion Analysis** вЂ” Every entry is analyzed by Claude AI for primary emotion, intensity, sentiment score, energy level, themes, and keywords
- **Visual Mood Canvas** вЂ” An interactive grid of colored tiles representing each entry's emotion вЂ” hover to preview, click to read
- **Mood Timeline Chart** вЂ” Line chart tracking your mood score over days
- **Emotion Radar** вЂ” Radar chart showing your emotional fingerprint over time
- **Weekly AI Insight Reports** вЂ” Personalized coaching, highlights, challenges, and an affirmation generated by Claude
- **Recurring Theme Clusters** вЂ” Word-cloud of themes detected across all your entries
- **Journaling Streaks** вЂ” Tracks consecutive days of journaling with a streak counter
- **JWT Authentication** вЂ” Secure register/login with bcrypt password hashing
- **Fully Responsive** вЂ” Works on desktop and mobile

## рџљЂ Getting Started

### Prerequisites

- Node.js 18+
- MongoDB (local or [MongoDB Atlas](https://www.mongodb.com/atlas))
- [GROQ API Key](https://console.groq.com)

### 1. Clone the repo

```bash
git clone https://github.com/pritam1011NP/moodcanvas
cd moodcanvas
```

### 2. Set up the backend

```bash
cd backend
cp .env.example .env
# Fill in your values in .env
npm install
npm run dev
```

**Backend `.env`:**
```env
PORT=5000
MONGO_URI=mongodb://localhost:27017/moodcanvas
JWT_SECRET=your_super_secret_jwt_key
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxx
CLIENT_URL=http://localhost:5173
```

### 3. Set up the frontend

```bash
cd ../frontend
npm install
npm run dev
```

### 4. Open the app

Visit [http://localhost:5173](http://localhost:5173)

## рџ”Њ API Reference

### Auth
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/auth/register` | Register new user |
| POST | `/api/auth/login` | Login |
| GET | `/api/auth/me` | Get current user |

### Journal
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/journal` | Create entry (triggers AI analysis) |
| GET | `/api/journal` | Get paginated entries |
| GET | `/api/journal/:id` | Get single entry |
| PUT | `/api/journal/:id` | Update entry |
| DELETE | `/api/journal/:id` | Delete entry |
| GET | `/api/journal/stats/timeline` | Get mood timeline data |

### Insights
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/insights/weekly` | AI weekly insight report |
| GET | `/api/insights/heatmap` | 3-month emotion heatmap |
| GET | `/api/insights/themes` | Recurring theme clusters |
